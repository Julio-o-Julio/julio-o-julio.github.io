Sobre mim

Me chamo Julio Souza e sou estudante de Engenharia de Software pela UFMS, moro atualmente em Campo Grande MS e estou em busca de uma posição no mercado. Possuo experiência em Desenvolvimento Web Back-end, construo aplicações com foco em JavaScript e Python com o Framework Django. No momento atuo em projetos pessoais para adquirir experiência e prática em resolução de problemas e pensamento lógico além de praticar as linguagens de programação e o Framework mencionado acima.

Experiência acadêmica

Na faculdade adiquiri base teórica em resolução de problemas e lógica de programação, coloco em prática usando as linguagens de programação Python; C e Java. Com isso tenho domínio em: Linguagens de baixo e alto nível; manipulação de banco de dados e versionamento de código.

Estudando

Estou me aperfeiçoando em JavaScript, Django, APIRest e banco de dados. Pois ter uma base forte em pelo menos uma linguagem de programação; Um framework; extrair dados de uma API e manipular um banco de dados é primordial para ser um Desenvolvedor Back-end.
