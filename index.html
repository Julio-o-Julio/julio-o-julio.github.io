Sobre mim

Me chamo Julio Souza e sou estudante de Engenharia de Software pela UFMS, resido atualmente em Campo Grande MS e estou em busca de uma posição no mercado. Possuo experiência em Desenvolvimento Web Back-end, construo aplicações com foco em JavaScript usando o Vite juntamente com o React; Python com o Framework Django. No momento atuo em projetos pessoais para adquirir experiência e prática em resolução de problemas e pensamento lógico além de praticar as linguagens de programação e os Frameworks mencionados acima.

Experiência acadêmica

Na faculdade adiquiri base teórica e prática em resolução de problemas e lógica de programação, coloco em prática usando as linguagens de programação: JavaScript; Python; C e Java. Para ficar organizado e prático, ultilizo o Git juntamente com o GitHub para armazenar o versionamento e histórico dos meus projetos. Com isso tenho domínio em: Linguagens de baixo e alto nível; manipulação de banco de dados; consumo de APIs e versionamento de código.

Estudando

Estou me aperfeiçoando em JavaScript, Django, APIRest e banco de dados. Pois ter uma base forte em pelo menos uma linguagem de programação; Um framework; extrair dados de uma API e manipular um banco de dados é primordial para ser um Desenvolvedor Back-end.
